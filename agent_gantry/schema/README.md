# agent_gantry/schema

Typed data models that define Agent-Gantry's public contracts.

- `__init__.py`: Schema package exports.
- `a2a.py`: Models for Agent-to-Agent (A2A) discovery and communication.
- `config.py`: Configuration models for gantry, embedders, vector stores, rerankers, telemetry, and servers.
- `events.py`: Telemetry event payloads and tracking schemas.
- `execution.py`: Tool call, result, and batch execution models.
- `query.py`: Query, retrieval result, and routing weight models.
- `tool.py`: Canonical tool definition, capabilities, and schema transcoding helpers.
